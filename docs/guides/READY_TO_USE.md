## 🚀 Quick Start (30 seconds)

### 1. Launch the App
```bash
python3 main.py
```

### 2. Try Text Replacement
1. Click **"Upload Image"**
2. Select **test_image.png** (already created for you!)
3. Press **`Ctrl+F`** (or click "🔍 Detect Text in Image")
4. Click any detected text in the list
5. Type your replacement text
6. Press **`Enter`** or click **"✏️ Replace Selected Text"**

**That's it!** The text is replaced automatically! 🎨
