## ⏳ Tesseract Installation (For Text Replacement Only)

### Option 1: Wait for Homebrew (Background)
Homebrew is installing in the background. Check if done:
```bash
brew list | grep tesseract
```

If installed, restart the app and text replacement will work!

### Option 2: Check Later
```bash
python3 check_tesseract.py
```

This will tell you if Tesseract is ready.

### Option 3: Use App Without It
99% of features work perfectly without Tesseract!
